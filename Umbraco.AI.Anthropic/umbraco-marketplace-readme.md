## Umbraco.AI.Anthropic

Anthropic provider for Umbraco.AI - integrate Claude models with extended context windows up to 200K tokens.

### Features

- **Claude Models** - Access Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Sonnet, and Claude 3 Haiku
- **Extended Context** - Support for Claude's industry-leading 200K token context windows
- **Chat Completions** - Streaming and non-streaming chat with all Claude models
- **Dynamic Model Discovery** - Automatically fetches available models from Anthropic API
- **Custom Endpoints** - Support for proxy servers or alternative endpoints

### Requirements

- Umbraco CMS 17.0.0+
- Umbraco.AI 1.0.0+
- .NET 10.0
- Anthropic API key
