## Umbraco.Ai.Google

Google provider for Umbraco.Ai - integrate Google's Gemini models with extended context windows up to 1M tokens.

### Features

- **Gemini Models** - Access Gemini 2.0 Flash, Gemini 1.5 Pro, Gemini 1.5 Flash, and variants
- **Extended Context** - Support for Gemini's massive context windows (up to 1M tokens on Pro)
- **Chat Completions** - Streaming and non-streaming chat with all Gemini models
- **Simple Setup** - Single API key configuration from Google AI Studio

### Requirements

- Umbraco CMS 17.0.0+
- Umbraco.Ai 1.0.0+
- .NET 10.0
- Google AI API key
