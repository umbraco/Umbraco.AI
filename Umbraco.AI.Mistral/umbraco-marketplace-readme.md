## Umbraco.AI.Mistral

Mistral provider for Umbraco.AI - connect directly to Mistral and use their chat and embedding models from the Umbraco backoffice.

### Features

- **Mistral Models** - Access Mistral Large, Small, Codestral, Pixtral, Ministral, Magistral, and the open-weight models
- **Chat Completions** - Streaming and non-streaming chat
- **Embeddings** - Generate vector embeddings with `mistral-embed`
- **Dynamic Model Discovery** - Automatically fetches available models from the Mistral API

### Requirements

- Umbraco CMS 17.0.0+
- Umbraco.AI 1.0.0+
- .NET 10.0
- Mistral API key
