## Umbraco.AI.Prompt

AI prompt template management for Umbraco.AI - create, organize, and execute reusable prompts with smart context injection.

### Features

- **Prompt Templates** - Store reusable prompts with {{variable}} placeholders and system instructions
- **Smart Scoping** - Target prompts to specific property editors, properties, or content types with allow/deny rules
- **Context Injection** - Automatically inject AI contexts and entity content into prompt execution
- **Profile Linking** - Associate prompts with AI profiles for consistent model and settings
- **One-Click Execution** - Run prompts directly from property editors with instant results
- **Property Actions** - Insert AI-generated content directly into content properties
- **Tag Organization** - Categorize and filter prompts with tags
- **Backoffice UI** - Full prompt management with scope builder and execution testing

### Requirements

- Umbraco CMS 17.0.0+
- Umbraco.AI 1.0.0+
- .NET 10.0
