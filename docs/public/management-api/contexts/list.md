# List Contexts

Returns a paginated list of all contexts.

## Request

```http
GET /umbraco/ai/management/api/v1/context
```

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `skip` | int | 0 | Number of items to skip |
| `take` | int | 100 | Number of items to return |
